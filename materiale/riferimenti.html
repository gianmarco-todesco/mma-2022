Linguaggio di programmazione

Useremo il linguaggio di programmazione JavaScript.

Il sito MDN Web Docs contiene una pagina interessante con tante informazioni e link a vari tutorial.

Il linguaggio è in continua evoluzione (ES5, ES6, ECMAScript 2018, etc.): noi useremo in genere ES6.

Oltre a Javascript ci servirà qualche accenno di HTML e CSS.

Browser

Per vedere le animazioni avremo bisogno di un Web Browser, ad esempio Google Chrome che è molto veloce nell'esecuzione del codice Javascript grazie ad un componente chiamato V8.

Anche altri browser vanno bene: ad esempio Mozilla Firefox.

Versioni non aggiornate di Internet Explorer o Safari possono dare dei problemi ed è meglio non usarle.

Editor

Per creare le animazioni dovremo scrivere dei file con l'estensione .html e .js. E' possibile utilizzare qualsiasi editor di testo. A lezione utilizzeremo Visual Studio Code, che è già installato sui PC del laboratorio. Il programma è gratuito e disponibile su Windows, Linux e macOS.

Il programma Visual Studio Code permette di installare delle estensioni. Per il lavoro che dovremo fare è molto comoda Live Server. Anche questa dovrebbe essere già installata. Diversamente la si può installare con la sequente procedura:

  • Lanciare Visual Studio Code e premere la combinazione di tasti ctrl-shift-x
  • Nel campo di ricerca, in alto a sx, scrivere Live Server. Il primo risultato dovrebbe essere quello giusto
  • Fare click sul risultato. Nel pannello di destra si può controllare se l'estensione è già installata oppure no.
  • Se l'estensione non è installata dovrebbe bastere premere il bottone installa

Tecnologia

Useremo WebGL in modo da controllare (dal programma javascript) la scheda grafica del computer su cui guardiamo l'animazione.

La tecnologia WebGL permette di realizzare animazioni veramente sorprendenti: qualche esempio:

  • WebGL Water
  • Traveling wavefronts
  • Cars
  • FractalLab
  • Jellyfish
  • ecc..

Framework

Per facilitare l'interazione con WebGL useremo una libreria 3D: Babylon.js (Manuale).

Nota bene: Ce ne sono altre, altrettanto valide. Ad es. Three.js, o Playcanvas.

Ispirazione per le tesine

L'argomento della tesina è libero. Le tesine verranno valutate da una giuria che valuterà i lavori sia sul piano tecnico che su quello estetico. Per ottenere una tesina memorabile è importante essere in grado di "raccontare" una storia, anche molto semplice. Per fissare le idee ecco tre tipologie di "storie":

  • Usare il 3D per spiegare un concetto complesso, ad es. un teorema di geometria oppure una legge fisica. Un altro esempio è mostrare un meccanismo complesso in azione, come ad es. il movimento di un motore a scoppio, con cilindri, pistoni, bielle e manovelle, ecc.
    In questo caso la chiarezza è cruciale.
  • Realizzare un gioco: o un gioco di logica oppure anche uno "sparatutto".
    Il gioco può essere semplice, ma deve essere coinvolgente: deve fare venire voglia di giocare.
  • Creare un'animazione soprendente, ad esempio basata su una coreografia complicata in cui sembra che gli oggetti si stiano per scontrare, ma all'ultimo momento si evitano per un soffio.
    In questo caso cerchiamo di stupire chi guarda.
